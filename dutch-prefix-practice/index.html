Mee or Aan? Placement Practice

How to Play

This game helps you practice using the Dutch prefixes "mee" and "aan" correctly in sentences.

For each sentence:

  1. Choose either "mee" or "aan" from the first dropdown.
  2. Decide if the prefix should be attached to the verb or placed at the end of the sentence.
  3. Click "Check Answer" to see if you're correct.

The game will provide feedback and explanations to help you learn!
